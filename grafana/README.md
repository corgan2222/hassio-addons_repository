# Home Assistant Community Add-on: Grafana Enterprise

[![Release][release-shield]][release] ![Project Stage][project-stage-shield] ![Project Maintenance][maintenance-shield]

The open platform for beautiful analytics and monitoring. Enterprise Edition.

## About

The analytics platform for all your metrics.

Grafana allows you to query, visualize, alert on and understand your metrics
no matter where they are stored. Create, explore, and share dashboards. Learn
about your Home Automation system using sexy and compelling graphs, and other
data visualizations.

This fork runs the **Enterprise Edition** of Grafana in a fully local,
self-hosted environment. Maintained by [Stefan Knaak (corgan2222)][corgan2222].
