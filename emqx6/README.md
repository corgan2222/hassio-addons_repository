# Home Assistant Community Add-on: EMQX6 Enterprise

[![Release][release-shield]][release] ![Project Stage][project-stage-shield] ![Project Maintenance][maintenance-shield]

EMQX Enterprise v6 — the most scalable open-source MQTT broker for IoT. Enterprise Edition.

## About

[EMQX][emqx] is an Open-source MQTT broker with a high-performance real-time
message processing engine, powering event streaming for IoT devices at massive
scale.

This fork runs the **Enterprise Edition v6** of EMQX in a fully local,
self-hosted environment. Maintained by [Stefan Knaak (corgan2222)][corgan2222].
